# Smart Parking Lot System (Java)

## Overview
This is a Java-based Smart Parking Lot System built using Object-Oriented Programming principles and SOLID design principles.
The system supports vehicle, parking space allocation, entry/exit tracking, payment processing, and notifications.

The application is designed to be easily extensible for advanced features such as multi-branch parking lots, and dynamic pricing.
